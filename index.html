CS 496 Cloud and Mobile Development

This is the course information page for CS496. This course provides an overview of mobile and cloud technologies. You will have a lot of freedom to explore the technologies that interest you in these areas. The class will focus on the fundamental concepts leaving much of the implementation to you.

You can find more information about the course by clicking the links above. It is suggested you start with the 'Overview'.
